## Type Styles in Figma

In the Figma library, you can find all SLDS type styles in the **Text styles** menu in the properties panel.

1. To apply a type style, highlight your text or select your text box. In the Text section of the right sidebar, click the Styles icon.
2. Choose the appropriate style. Text styles have labels based on their`font-scale` styling hook names and are listed with the available weights for each size.

To apply a type style, highlight your text or select your text box. In the Text section of the right sidebar, click the Styles icon.

ℹ️ Learn more about how to apply [Type Styles in Figma](https://help.figma.com/hc/en-us/articles/360039957034-Create-and-apply-text-styles#apply).

### Type styles tips

- All SLDS type styles are tied to Figma variables, which surface in code as style hooks.
- Don't use the Variable options to change your type selection.
- Hover over a text style to see a description of where it’s used.
