## Usage

- Use connectors to connect to nodes horizontally or vertically.
- Avoid 45° and freeform connectors.
- Use the badge component to label connectors.
In testing mode, a user can choose a path through a workflow and then see it reflected in color and line-weight changes to the connector path.
