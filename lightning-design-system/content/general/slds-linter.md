## Introducing SLDS Linter

SLDS Linter analyzes your code against SLDS 2 rules. It offers suggestions on how to improve your code and make it more future-proof. SLDS Linter can also automatically fix your code and apply those suggestions in bulk across your repository.

As a developer or designer, you’re aware of SLDS Validator, your tool to assess your code base and update your code to SLDS 1 standards and style guidelines. Now with SLDS 2, you need a tool that’s simpler to set up, and is flexible and easy to use while transitioning your org to SLDS 2.

SLDS Linter is the latest code analysis tool that helps smooth your org’s transition to SLDS 2 and future-proofs your code.

SLDS Validator is still available for your use, but SLDS Linter provides many advantages. SLDS Linter is easier to set up, offers increased automation, and contains a more robust library of rules reflecting the new SLDS 2 styling guidelines.

With just two Terminal commands, you can easily install and run SLDS Linter. SLDS Linter analyzes your code against the built-in SLDS 2 rules engine, reports detailed findings, and suggests fixes to ensure compliance with SLDS 2 styling guidelines. Taking it one step further, SLDS Linter reduces manual uplift effort by applying the suggested fixes in bulk across your repository.
