## Doctype Icons

[Browse Icons Browse the SLDS icon library](https://www.lightningdesignsystem.com/2e1ef8501/v/0/p/83309d-icons/b/586464)

Doctype iconsrepresent file types, and are often used when a file preview is unavailable. Use a recognizable white glyph inside these icons if a familiar visual metaphor is available; otherwise, use a text abbreviation of the corresponding file extension.

Doctype icons are used in both mobile and desktop environments, including inside feeds, publishers, cards, and related lists.

### Anatomy

- Based on a grid of 56x64px
- Live area of 36x12px (letter) or 36x22px (icon), with 10px padding
- 6px corner radius
- Icon: color #fff
- Text: Proxima Nova Soft, Regular 18 pt, color #fff
- Flap: color #fff with opacity of 50%; drop shadow: color #000 with opacity of 10%
- Add a background color


### Keyline Shape

Vertical Rectangle (56x64px)

### Artboard

- Recommended canvas size for setting up Illustrator or Figma files: 56x64px
- Live area of 36x12px (letter) or 36x22px (icon), with 10px padding
