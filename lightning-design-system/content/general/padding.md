Padding is the space between a component's borders and its contents. Padding keeps content from getting crowded and helps maintain a clean, readable layout.

A note on padding before use:

- The base unit for all SLDS spacing metrics is **4**.
- Most components have spacing included. These utility classes are for added convenience in laying out components.
- Classes prefixed by `slds-**p**-` are used to add padding to an element. Classes prefixed by `slds-**m**-` are used to add [margin](https://www.lightningdesignsystem.com/2e1ef8501/p/51dd56) to an element.
- Spacing class names use these direction indicators:`top`, `right`, `bottom`, and `left`.
- Use the shortcut `vertical` to apply space to both `top` and `bottom`.
- Use the shortcut `horizontal` to apply space to both `right` and `left`.
- Use the shortcut `around` to apply space to all sides.
- Use the `_xxx-small` through `_xx-large` scale to choose the spacing size needed.
- Use `none` as the size on any of the directions to enforce zero padding.
- For vertical centering, check out our [centered media object](https://www.lightningdesignsystem.com/2e1ef8501/p/87b07a) instead of applying extra padding or margin.
