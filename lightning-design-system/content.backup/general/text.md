Text utility classes provide a simple way to make text look consistent across Salesforce applications. These classes help developers and designers apply typography-related styling and alignment to ensure text is consistent and clear.

When building an enterprise application, the heading level can vary depending on the context of the component or page. The text classes in our framework start at the base (body text) size and scale up for headings (h1-h6), with no default margins or padding set to zero. To maintain accessibility, use the correct heading level.
