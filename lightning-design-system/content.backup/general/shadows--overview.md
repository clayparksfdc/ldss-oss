## About

Shadow and elevation play a vital role in the visual design of user interfaces. They simulate depth and dimension, which establishes a sense of hierarchy and depth perception. Shadows and elevations dictate how elements interact with light and space, ultimately shaping the aesthetics and user experience.

## Box Shadow

The boxshadowCSS property adds a shadow effect to an element. This property sets values for horizontal and vertical offsets, blur radius, spread radius, and shadow color. The combination of properties creates a shadow around the frame of an element. Theshadowproperty is widely used to add depth and dimension to elements on a web page through various shadow effects.

Elevation is applied to elements to show that surfaces can move on top of one another.

The SLDS 2 design uses soft shadows to create a sense of depth and dimension in the user interface. They also help separate components from each other and create a more realistic look.

The styling hook for shadows uses the label shadow.
