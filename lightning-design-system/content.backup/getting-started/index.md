# Getting Started

Welcome to the Lightning Design System! This guide will help you get up and running quickly.

::: callout type="info"
The Lightning Design System provides a comprehensive set of components and guidelines for building Salesforce applications.
:::

## Installation

To install the Lightning Design System in your project, run:

```bash
npm install @salesforce-ux/design-system
```

## Basic Usage

Import the CSS in your application:

```javascript
import '@salesforce-ux/design-system/assets/styles/salesforce-lightning-design-system.min.css';
```

::: callout type="success"
You're now ready to start using Lightning Design System components!
:::
