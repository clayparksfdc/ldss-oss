## Examples

### Conditional

Only one section can be expanded at a time.

Use Case: Use to help the user concentrate on one section at a time.

### Multiple

Allows multiple sections to be open at once.

Use Case: When users need to reference more than one section at a time.

### Action Menu

The optional actions slot provides additional actions related to the accordion section through an overflow menu. While various button types can be used in this slot, we recommend using Button Menu component to maintain a consistent experience.
