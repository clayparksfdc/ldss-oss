## Border Styling Hooks

All border styling hooks are prefixed with `--slds-g-` and are followed by a border name property. For example, --slds-g-sizing-border-1 (1px) is the styling hook for a border width of 1 px.

### Width Hooks

Use these styling hooks to specify the pixel width of a border.

![Global Boarders](https://zeroheight-uploads.s3.eu-west-1.amazonaws.com/03e3c33ad048391f34a4db?X-Amz-Algorithm=AWS4-HMAC-SHA256&X-Amz-Credential=AKIA3AVNYHQK6SJI2PWU%2F20260518%2Feu-west-1%2Fs3%2Faws4_request&X-Amz-Date=20260518T042850Z&X-Amz-Expires=604800&X-Amz-SignedHeaders=host&X-Amz-Signature=1b2d409c04b034fa30a693fbcf4d0c26d9c5ee13766229a49cfb08ad64a72161)

**Global Boarders**

---

| Token | Value | Description |
| --- | --- | --- |
| --slds-g-sizing-border-1 | 1px | Border Width 1 | 1px. |
| --slds-g-sizing-border-2 | 2px | Border Width 2 | 2px. |
| --slds-g-sizing-border-3 | 3px | Border Width 3 | 3px. |
| --slds-g-sizing-border-4 | 4px | Border Width 4 | 4px. |

### Radius Hooks

These properties manage the curvature of UI elements' corners, impacting the perceived softness or hardness of components.

![Global Radius](https://zeroheight-uploads.s3.eu-west-1.amazonaws.com/951ed516ca42a67de758e5?X-Amz-Algorithm=AWS4-HMAC-SHA256&X-Amz-Credential=AKIA3AVNYHQK6SJI2PWU%2F20260518%2Feu-west-1%2Fs3%2Faws4_request&X-Amz-Date=20260518T042850Z&X-Amz-Expires=604800&X-Amz-SignedHeaders=host&X-Amz-Signature=415bf1988f1d0c2ae0009ae6a9fa1acb87eda2d8206366dde5c6b0da8f5de732)

**Global Radius**

---

| Token | Value | Description |
| --- | --- | --- |
| --slds-g-radius-border-1 | 0.25rem | Radius Border 1 | 4px. |
| --slds-g-radius-border-2 | 0.5rem | Radius Border 2 | 8px. |
| --slds-g-radius-border-3 | 0.75rem | Radius Border 3 | 12px. |
| --slds-g-radius-border-4 | 1.25rem | Radius Border 4 | 20px. |
| --slds-g-radius-border-circle | 100% | Radius Border Circle | 100% 100px. |
| --slds-g-radius-border-pill | 15rem | Radius Border Pill. |
