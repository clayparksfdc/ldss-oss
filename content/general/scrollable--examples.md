Scrollable utility classes help manage overflow content within fixed-sized containers, when scrolling is available. These classes control whether elements are scrollable and define the type of scrolling—horizontal, vertical, or both.

Use the scrollable utility classes to provide scrolling within a section of a page. For example, apply `slds-scrollable_y` to display a vertical scrollbar if the content within an element exceeds the height of the element. Apply  `slds-scrollable_x` to display a horizontal scrollbar if content within an element exceeds the width of the element.
