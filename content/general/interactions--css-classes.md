**Property**

**Description**

Selector

The CSS class being referred to.

Summary

A description of what the class does.

Support

Support can be either dev-ready or prototype.

- dev-ready class names are fully vetted, tested, and safe to use.

- prototype class names haven't been fully vetted yet.

Restrict

The selector that the class name can be used on.

Variant

The base-level pattern for a component. A variant can be extended to create another variant of that component, for example, a stateful button is a derivative of the base button.

Modifier

A single class that can be added to an HTML element of a component to modify its output. Typically these are colors, sizing, and positioning classes.
