Box utility classes help apply consistent borders and spacing for box styling to elements. These utilities simplify the process of visually defining containers and content areas within an application.

A [box theme](https://www.lightningdesignsystem.com/2e1ef8501/p/11cfec) can be added to the box to apply a background color or texture.
