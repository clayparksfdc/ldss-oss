Layout utility classes provide tools for structuring page and component layouts with consistent alignment, positioning, and flexibility. These classes help establish predictable and responsive designs that adapt to varying screen sizes and user needs.

## Examples

### Magnets

The magnet utilities are used to vertically attach adjacent card-like components.

These components/utilities provide a card-like look and are good use cases for magnet utilities.

To align a card-like component flush to another card-like component positioned below it, add the class `slds-has-bottom-magnet` to the top component and `slds-has-top-magnet` to the bottom component. In the After example, the adjacent edges of the cards are flattened and appear connected.
