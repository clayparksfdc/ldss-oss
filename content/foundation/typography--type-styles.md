A type style is a combination of font scale, weight, and line height. SLDS has three type styles: body, title, and display. Each type style serves a specific purpose.

## Body Type

Text that conveys details in the form of phrases, labels, sentences, or blocks of copy.

- Bold weight is used for emphasis only, not headings.
- --slds-g-color-on-surface-3

 is only used for headings and titles.
- All font weights and sizes remain the same in any system font.

## Title Type

Headings of components or body content.

## Display Type

Short titles in banners or prominent sections to make a bold visual statement.
