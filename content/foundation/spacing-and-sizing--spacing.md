## About

Spacing controls the empty areas around or within components, such as margins, padding, and gaps between elements. Effective spacing improves readability, reduces visual clutter, and enhances usability.

Spacing defines these visual aspects of a page.

- Proper alignment of components.
- Clear differentiation of grouped and unrelated elements.
- White space that directs user attention to key content or actions.
Spacing in styling hooks refers to padding or margins around an element.

## Spacing Scale

The spacing values create space between elements. The styling hook values are relative to the root font size and follow a modular scale of 4. Use `spacing` styling hooks to set values for margins and padding, for example.

Don't use spacing properties to establish dimensions of an element targeting width and height. Use the sizing properties for this purpose.

SLDS 2 uses the same spacing values as SLDS, with multiples of 4 to align with the 4-point grid system.

## Understanding Padding and Margin

Padding and margin are two different design elements. Padding controls the internal spacing within a component, while the margin defines the external spacing around it. Design systems follow a consistent margin strategy so that components interact predictably and maintain harmonious spacing throughout the interface.
